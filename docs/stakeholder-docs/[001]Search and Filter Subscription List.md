# Feature: Subscription Search

# Overview
We need to implement a search and filter feature. This will help the user find the subscription on the list.

# Business Need
User has a lot of subscriptions on the list, we need to implement a search and filter to help them find the subscription faster.

# Requirements
- Search subscriptions by subscription name
- Filter subscriptions by cost range
- Display correct data or 'No Results'
- Combined Search and Filter
