- ✓ Message visible for at least 2-3 seconds
- ✓ Message auto-dismisses or user can dismiss it

- ✓ Appears in SubscriptionList immediately (no page refresh)
- ✓ Displayed with Name, Cost (currency formatted), Due Date, Edit/Delete buttons

### AC6: Subscription Persists After Page Refresh
- ✓ Remains in list after reload
- ✓ Subscription data intact (name, cost, dueDate, ID, timestamps)
- ✓ No data loss or corruption
- ✓ Loaded from localStorage (handled by Story 2.5)

### AC7: Error Handling for Invalid Form Submission
- ✓ Form does NOT submit if required field missing (React Hook Form)
- ✓ Shows validation error for each invalid field
- ✓ No error action dispatched

### AC8: Form Data Types Are Correct
- ✓ name: string
